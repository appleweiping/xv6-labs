== xv6 Lab: mmap (memory-mapped files) — grade-lab-mmap ==
Host: WSL2 Ubuntu, QEMU riscv64 TCG emulation.
Implementation written from scratch (kernel/sysmmap.c + VMA in proc.h,
lazy-fault handling in trap.c, uvmunmap_lazy/walk_lookup in vm.c):
- mmap() records a VMA (lazy); regions grow downward below TRAPFRAME.
- page fault (scause 13/15) -> mmap_fault(): kalloc, readi the file slice,
  map with prot-derived PTE bits.
- munmap(): flush dirty MAP_SHARED pages via PTE_D, uvmunmap_lazy the range,
  shrink/close the VMA; supports prefix and suffix unmap.
- fork copies VMAs (filedup, lazy re-fault in child); exit runs mmap_cleanup.
